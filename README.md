# 💰 Smart Expense Tracker

## Overview

Smart Expense Tracker is a web-based personal finance management application that helps users track income and expenses, monitor balances, categorize transactions, and gain financial insights through interactive visualizations.

The application allows users to manage daily transactions efficiently while providing real-time summaries and spending analytics to improve budgeting and financial planning.

---

## Features

### Transaction Management
- Add income and expense transactions
- Categorize expenses
- Delete transactions
- Real-time balance calculation

### Financial Analytics
- Income tracking
- Expense tracking
- Balance monitoring
- Interactive pie chart visualization

### Data Persistence
- LocalStorage integration
- Automatic data saving
- Persistent transaction history

### User Experience
- Clean and responsive interface
- Real-time updates
- Easy transaction management
